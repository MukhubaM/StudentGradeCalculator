# Student Grade Calculator

##  Overview
The **Student Grade Calculator** is a simple Java application that helps calculate and manage student grades.  
It demonstrates core programming concepts such as input handling, conditional logic and basic arithmetic operations.

---

##  Features
- **Add Student Grades** – Enter marks for subjects.  
- **Calculate Final Grade** – Compute average and assign grade (A–F).  
- **Display Results** – Show student details with calculated grade.  
- **Simple Console UI** – Easy to use and beginner‑friendly.  

---

##  Tech Stack
- **Java (JDK 17+)**
- **Object-Oriented Programming**
- **Console-based UI**

---

##  Project Structure
- `Student.java` → Model class holding student details and marks.  
- `GradeCalculator.java` → Logic for calculating averages and grades.  
- `Main.java` → Entry point to run the application.
